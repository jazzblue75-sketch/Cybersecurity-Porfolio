# TryHackMe: Offensive Security Intro

## Objective
Learn basic offensive security concepts and use tools to discover hidden directories on a web server.

## Tools Used
- Dirb

## What I Did
- Used Dirb to scan a target web application
- Discovered hidden directories not visible through normal browsing
- Navigated to a hidden page that allowed account interaction
- Successfully modified account data by adding funds

## Key Concepts Learned
- Directory brute forcing
- Importance of hidden endpoints in web security
- How attackers discover unlinked or sensitive pages

## Why This Matters
Hidden directories can expose sensitive functionality. If not properly secured, attackers can manipulate data or gain unauthorized access.

## Skills Practiced
- Web enumeration
- Basic offensive security techniques
- Using Dirb for directory discovery
